Purpose

The purpose of this web site is to demonstrate the mobile web, what features are available and what you need to consider when developing web solutions for mobile devices. The web pages and code here is only a compliment to books or other educational surroundings since there is not much text explaining what is shown.

Besides touching (or clicking) around the site and trying out web pages, perhaps in different devices to see what might differ, you are also supposed to "View Source" and check out the html, css and JavaScript that made it all possible. That's why the source is also published on GitHub

Note that although the site was made for the mobile web, all web pages are compatible with those old devices known as "computers".
